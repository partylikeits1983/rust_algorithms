# rust_practice

Learning Rust by completing HackerRank & Leetcode problems

### Compile using cargo:  
Change target path in Cargo.toml file 
```
path = "src/rotate_left/main.rs"
```

Run program
```
cargo run
```


### Compile using rustc: 
```
rustc src/array_sum/main.rs
```

to run:
```
./main
```
